# Image_Generation_Project
Create photos usingOpenAI API

The CustomCTKinter project is a simple desktop application with couple of buttons, radiobuttons and textbox entry. What it does is download and display generated by openAi DALL-E AI image after user input of the description of the photo desired.

First use of 
- CustomTKinter,
- dotenv lib - unsing .env files to safekeep sensitive info like passwords and API keys, etc.
- io lib, BytesIO - used to handle downloading of information to RAM
- openai - DALL-E API to generate images
- urllib - to open urls
- PIL library - used to load images
